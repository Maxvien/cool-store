# CoolStore

CoolStore is a simple and powerful state store built on top of [ImmerJS](https://www.npmjs.com/package/immer) and [RxJS](https://www.npmjs.com/package/rxjs).

## Installation

```
npm install --save rxjs
```

```
npm install --save cool-store
```
